Web and Internet Technology:COM1008

This module is for students studying Computer Science as a main subject. The Web and the Internet are now pervasive in modern life, providing an information resource, supporting ‘cloud computing’, and promoting novel, interactive solutions to computing-related activities. This module will cover the basics of networks and internetworking, the Web in society, including information security and the technologies that are used in Web development, with an emphasis on building a simple interactive website using HTML, CSS and JavaScript COM1008 module page
